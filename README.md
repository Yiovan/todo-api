# Proyecto Todo App

Este proyecto es una aplicación simple de lista de tareas con backend en PHP y base de datos PostgreSQL.

## Requisitos
- PHP >= 7.4
- PostgreSQL
- Servidor local como XAMPP, WAMP o Docker
